# LogiTalk

***LogiTalk*** - це простий та зручний онлайн чат з підтримкою обміном повідомленнями і зображеннями в реальному часі. Побудований на Python, Tkinter, Customtkinter, сокетах.

## Можливості:
- Надсилання повідомлень
- Надсилання зображень
- Простий інтерфейс
- Підтримка кількох користувачів

## Використані технології:
- socket
- base64
- Pillow (PIL)
- Pyton
